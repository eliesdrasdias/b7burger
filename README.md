# 🍔 B7Burger - E-commerce Responsivo

Uma interface digital completa e responsiva para uma hamburgueria, focada em conversão e experiência do usuário (UX/UI). 

## 💻 Sobre o Projeto
Este projeto foi desenvolvido para simular o front-end de um sistema de delivery de uma hamburgueria virtual. O foco principal foi criar um layout fluido que se adapta perfeitamente a dispositivos móveis, tablets e desktops, garantindo que o cliente consiga visualizar o cardápio e fazer pedidos de qualquer tela.

## 🛠️ Tecnologias Utilizadas
* **HTML5 Semântico:** Estruturação limpa e acessível.
* **CSS3:** Estilização moderna.
* **Flexbox & CSS Grid:** Construção do layout, alinhamentos e da grade de produtos de forma responsiva.
